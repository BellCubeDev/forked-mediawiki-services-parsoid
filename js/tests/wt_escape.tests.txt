#### --------------- Headings ---------------
#### 0. Unnested
#### 1. Nested inside html <h1>=foo=</h1>
#### 2. Outside heading nest on a single line <h1>foo</h1>*bar
#### 3. Nested inside html with wikitext split by html tags
#### 4. No escape needed
#### ----------------------------------------
!! test
Headings: 0. Unnested
!! input
<nowiki>=foo=</nowiki>
<nowiki>=foo</nowiki>''''=
!! result
=foo=
=foo<i></i>=
!!end

!! test
Headings: 1. Nested inside html 
!! input
=<nowiki>=foo=</nowiki>=
==<nowiki>=foo=</nowiki>==
===<nowiki>=foo=</nowiki>===
====<nowiki>=foo=</nowiki>====
=====<nowiki>=foo=</nowiki>=====
======<nowiki>=foo=</nowiki>======
!! result
<h1>=foo=</h1>
<h2>=foo=</h2>
<h3>=foo=</h3>
<h4>=foo=</h4>
<h5>=foo=</h5>
<h6>=foo=</h6>
!!end

!! test
Headings: 2. Outside heading nest on a single line <h1>foo</h1>*bar
!! input
=foo=
<nowiki>*bar</nowiki>
=foo=
=bar
=foo=
<nowiki>=bar=</nowiki>
!! result
<h1>foo</h1>*bar
<h1>foo</h1>=bar
<h1>foo</h1>=bar=
!!end

!! test
Headings: 3. Nested inside html with wikitext split by html tags 
!! input
=<nowiki>=</nowiki>''''''foo==
=<nowiki>=</nowiki>'''bold'''foo==
!! result
<h1>=<b></b>foo=</h1>
<h1>=<b>bold</b>foo=</h1>
!!end

!! test
Headings: 4. No escaping needed (testing just h1 and h2)
!! input
==foo=
=foo==
===foo==
==foo===
=''=''foo==
===
!! result
<h1>=foo</h1>
<h1>foo=</h1>
<h2>=foo</h2>
<h2>foo=</h2>
<h1><i>=</i>foo=</h1>
<h1>=</h1>
!!end

#### --------------- Lists ---------------
#### 0. Outside nests (*foo, etc.)
#### 1. Nested inside html <ul><li>*foo</li></ul>
#### 2. Inside definition lists
#### 3. Only bullets at start should be escaped
#### ----------------------------------------

!! test
Lists: 0. Outside nests 
!! input
<nowiki>*foo</nowiki>
<nowiki>#foo</nowiki>
!! result
*foo
#foo
!!end

!! test
Lists: 1. Nested inside html
!! input
*<nowiki>*foo</nowiki>
*<nowiki>#foo</nowiki>
*<nowiki>:foo</nowiki>
*<nowiki>;foo</nowiki>
#<nowiki>*foo</nowiki>
#<nowiki>#foo</nowiki>
#<nowiki>:foo</nowiki>
#<nowiki>;foo</nowiki>
!! result
<ul><li>*foo</li></ul>
<ul><li>#foo</li></ul>
<ul><li>:foo</li></ul>
<ul><li>;foo</li></ul>
<ol><li>*foo</li></ol>
<ol><li>#foo</li></ol>
<ol><li>:foo</li></ol>
<ol><li>;foo</li></ol>
!!end

!! test
Lists: 2. Inside definition lists
!! input
;<nowiki>;foo</nowiki>
;<nowiki>:foo</nowiki>
;<nowiki>:foo</nowiki>
:bar
:<nowiki>:foo</nowiki>
!! result
<dl><dt>;foo</dt></dl>
<dl><dt>:foo</dt></dl>
<dl><dt>:foo</dt><dd>bar</dd></dl>
<dl><dd>:foo</dd></dl>
!!end

!! test
Lists: 3. Only bullets at start of text should be escaped
!! input
*<nowiki>*foo*bar</nowiki>
*<nowiki>*foo</nowiki>''it''*bar
!! result
<ul><li>*foo*bar</li></ul>
<ul><li>*foo<i>it</i>*bar</li></ul>
!!end

#### --------------- Links ---------------
#### 1. Quote marks in link text
#### 2. Wikilinks: Escapes needed
#### 3. Wikilinks: No escapes needed
#### 4. Extlinks: Escapes needed
#### 5. Extlinks: No escapes needed
#### --------------------------------------
!! test
Links 1. Quote marks in link text
!! input
[[Foo|<nowiki>Foo''boo''</nowiki>]]
!! result
<a rel="mw:WikiLink" href="Foo" data-parsoid="{&quot;tsr&quot;:[0,7],&quot;contentPos&quot;:[5,5],&quot;src&quot;:&quot;[[Foo]]&quot;,&quot;bsp&quot;:[0,7],&quot;stx&quot;:&quot;simple&quot;}">Foo''boo''</a>
!! end

!! test
Links 2. WikiLinks: Escapes needed
!! input
[[Foo|<nowiki>[Foobar]</nowiki>]]
[[Foo|<nowiki>Foobar]</nowiki>]]
[[Foo|<nowiki>x [Foobar] x</nowiki>]]
[[Foo|<nowiki>x [http://google.com g] x</nowiki>]]
[[Foo|<nowiki>[[Bar]]</nowiki>]]
[[Foo|<nowiki>x [[Bar]] x</nowiki>]]
[[Foo|<nowiki>|Bar</nowiki>]]
!! result
<a href="Foo" rel="mw:WikiLink">[Foobar]</a>
<a href="Foo" rel="mw:WikiLink">Foobar]</a>
<a href="Foo" rel="mw:WikiLink">x [Foobar] x</a>
<a href="Foo" rel="mw:WikiLink">x [http://google.com g] x</a>
<a href="Foo" rel="mw:WikiLink">[[Bar]]</a>
<a href="Foo" rel="mw:WikiLink">x [[Bar]] x</a>
<a href="Foo" rel="mw:WikiLink">|Bar</a>
!! end

!! test
Links 3. WikiLinks: No escapes needed
!! input
[[Foo|[Foobar]]
[[Foo|foo|bar]]
!! result
<a href="Foo" rel="mw:WikiLink">[Foobar</a>
<a href="Foo" rel="mw:WikiLink">foo|bar</a>
!! end

!! test
Links 4. ExtLinks: Escapes needed
!! input
[http://google.com <nowiki>[google]</nowiki>]
[http://google.com <nowiki>google]</nowiki>]
!! result
<a href="http://google.com" rel="mw:ExtLink">[google]</a>
<a href="http://google.com" rel="mw:ExtLink">google]</a>
!! end

!! test
Links 5. ExtLinks: No escapes needed
!! input
[http://google.com [google]
!! result
<a href="http://google.com" rel="mw:ExtLink">[google</a>
!! end

#### --------------- Quotes ---------------
#### 1. Quotes inside <b> and <i>
#### 2. Link fragments separated by <i> and <b> tags
#### 3. Link fragments inside <i> and <b>
#### --------------------------------------
!! test
1. Quotes inside <b> and <i>
!! input
''<nowiki>'foo'</nowiki>''
''<nowiki>''foo''</nowiki>''
''<nowiki>'''foo'''</nowiki>''
'''<nowiki>'foo'</nowiki>'''
'''<nowiki>''foo''</nowiki>'''
'''<nowiki>'''foo'''</nowiki>'''
'''<nowiki>foo'</nowiki>''<nowiki>bar'</nowiki>''baz'''
!! result
<i>'foo'</i>
<i>''foo''</i>
<i>'''foo'''</i>
<b>'foo'</b>
<b>''foo''</b>
<b>'''foo'''</b>
<b>foo'<i>bar'</i>baz</b>
!! end

!! test
2. Link fragments separated by <i> and <b> tags
!! input
<nowiki>[[</nowiki>''''<nowiki>hello]]</nowiki>
<nowiki>[[</nowiki>''foo''<nowiki>hello]]</nowiki>
<nowiki>[[</nowiki>'''foo'''<nowiki>hello]]</nowiki>
!! result
[[<i></i>hello]]
[[<i>foo</i>hello]]
[[<b>foo</b>hello]]
!! end

!! test
2. Link fragments inside <i> and <b>
(FIXME: Escaping one or both of [[ and ]] is also acceptable -- 
 this is one of the shortcomings of this format)
!! input
''[[foo''<nowiki>]]</nowiki>
'''[[foo'''<nowiki>]]</nowiki>
!! result
<i>[[foo</i>]]
<b>[[foo</b>]]
!! end

#### --------------- HTML tags ---------------
#### 1. a tags
#### 2. other tags
#### 3. multi-line html tag
#### --------------------------------------
!! test
1. a tags
!! input
<a href="http://google.com">google</a>
!! result
&lt;a href=&quot;http://google.com&quot;&gt;google&lt;/a&gt;
!! end

!! test
2. other tags
!! input
<nowiki><div>foo</div></nowiki>
<nowiki><div style="color:red">foo</div></nowiki>
!! result
&lt;div&gt;foo&lt;/div&gt;
&lt;div style=&quot;color:red&quot;&gt;foo&lt;/div&gt;
!! end

!! test
3. multi-line html tag
!! input
<nowiki><div</nowiki>
<nowiki>>foo</div</nowiki>
<nowiki>></nowiki>
!! result
&lt;div
&gt;foo&lt;/div
&gt;
!! end
